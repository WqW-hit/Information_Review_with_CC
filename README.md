# Information_Review_with_CC
# 信息论复习资料

本仓库收录哈工深信息论课程（Lecture 2–16）的完整复习资料，共两类文件：

- **知识清单**：教授视角的讲义式 HTML，含公式、重要程度标注、常见易错点
- **章节测试**：交互式 HTML 测验，可即时核对答案与解析

## 使用方式

将仓库 Clone 到本地后，直接用浏览器打开任意 `.html` 文件即可。文件内嵌 CSS 与 JavaScript，**无需联网**，MathJax 公式在初次打开时需要网络加载（后续可离线）。
