1. 信源-信道分离编码（Source-Channel Separation）

1.1 系统架构与分离思想 必考

信源-信道分离编码将信源压缩与信道传输解耦为两个独立模块：

  • 信源编码（压缩器）：将 $N$ 长信源序列 $V^N$ 压缩为 $m = \log_2 M$ bits，要求 $m \geq NR^I(D)$
  • 信道编码（传输器）：将 $m$ bits 经 $n$ 次信道传输，要求 $m \leq nC^I$

关键条件：若 $NR^I(D) \leq nC^I$，则分离编码方案存在，选取满足 $$NR^I(D) \leq m \leq nC^I$$ 的状态数 $M$，分别设计最优压缩与最优信道编码即可实现端到端失真 $\leq D$。

易错：分离方案要求 $NR^I(D) \leq nC^I$，即 rate $= N/n \leq C^I/R^I(D)$，而非简单的 $R^I(D) \leq C^I$。

1.2 联合编码 vs 分离编码 必考

信源-信道联合编码使用统一编码器直接将 $V^N$ 映射到 $X^n$，rate $= N/n$。

可行性约束推导（马尔可夫链 $V^N \to X^n \to Y^n \to \hat{V}^N$）：

  • 噪声信道编码定理（反向）：$I(X^n; Y^n) \leq nC^I$
  • 率失真定理（反向）：$I(V^N; \hat{V}^N) \geq NR^I(D)$
  • 数据处理不等式：$I(V^N; \hat{V}^N) \leq I(X^n; Y^n)$

合并：$NR^I(D) \leq I(V^N; \hat{V}^N) \leq I(X^n; Y^n) \leq nC^I$

因此 分离方案不损失最优性，联合编码无法超越分离编码的理论极限。

易错：联合编码并不比分离编码更好。信源信道分离定理保证分离方案等价于最优联合方案。

1.3 分离定理结论 必考

定理：对独立同分布信源 $V_i \sim V$ 与离散无记忆信道，以 rate $= N/n$ 传输且失真 $\leq D$ 的方案存在，当且仅当： $$\frac{N}{n} \leq \frac{C^I}{R^I(D)}$$

实践意义：可分别独立设计最优压缩编码（达到 $R^I(D)$）与最优信道编码（达到 $C^I$），串联即构成最优端到端系统。

2. 变分自编码器（VAE）

2.1 率失真三种视角 重点

  • 失真-率问题：最小化失真，约束 rate $\leq R$
  • 率-失真问题（Shannon）：最小化 rate，约束 distortion $\leq D$
  • 联合最小化（VAE）：$\min\ \text{Rate} + \text{Distortion}$

VAE 的损失函数正是第三种视角：重建项（失真度）+ KL 散度项（压缩率）。

2.2 变分下界（ELBO）推导 重点

目标：最大化对数似然 $\log p(x)$。引入变分后验 $q(z|x)$： $$\log p(x) = \mathbb{E}_{q(z|x)}\!\left[\log \frac{p(z,x)}{q(z|x)}\right] + D_{\mathrm{KL}}(q(z|x)\|p(z|x)) \geq \text{ELBO}$$

ELBO 的等价形式： $$\text{ELBO} = \mathbb{E}_{q(z|x)}[\log p(x|z)] - D_{\mathrm{KL}}(q(z|x)\|p(z))$$

最大化 ELBO 等价于最小化 loss $= -\text{ELBO} = \underbrace{\mathbb{E}[-\log p_\theta(\hat{x}|z)]}_{\text{失真/重建}} + \underbrace{D_{\mathrm{KL}}(q_\phi(z|x)\|p(z))}_{\text{压缩率/正则}}$

易错：ELBO 是 $\log p(x)$ 的下界（非上界）。因为 $D_{\mathrm{KL}} \geq 0$，所以 $\log p(x) \geq \text{ELBO}$。

2.3 VAE 结构与参数 重点

编码器：$q_\phi(z|x) = \mathcal{N}(\mu, \mathrm{diag}(\sigma^2))$，输出 $(\mu, \log\sigma^2)$

重参数化技巧：$z = \mu + \sigma \odot \epsilon,\quad \epsilon \sim \mathcal{N}(\mathbf{0}, \mathbf{I})$ （使采样可微，允许梯度反向传播）

正则项闭式解： $$D_{\mathrm{KL}}(q_\phi(z|x)\|p(z)) = \frac{1}{2}\sum_i \left(\mu_i^2 + \sigma_i^2 - \ln\sigma_i^2 - 1\right)$$

重建项（$p_\theta(\hat{x}|z) = \mathcal{N}(\hat{x}, \sigma_c^2 I)$）： $$\mathbb{E}[-\log p_\theta(\hat{x}|z)] = \frac{\|x - \hat{x}\|^2}{\sigma_c^2} = \text{MSE}$$

易错：重参数化不是为了加速，而是为了使采样操作对 $\phi$ 可微，从而实现端到端反向传播。

2.4 正则项的信息论含义 常考

正则项分解： $$\mathbb{E}_{x \sim p_{data}}[D_{\mathrm{KL}}(q_\phi(z|x)\|p(z))] = I(X;Z)_{q_\phi} + D_{\mathrm{KL}}(q_\phi(z)\|p(z)) \geq I(X;Z)$$

含义：正则项是互信息 $I(X;Z)$（压缩率）的上界，同时惩罚聚合后验 $q_\phi(z)$ 偏离先验 $p(z)$ 的额外编码开销。

2.5 beta-VAE 常考

引入权重 $\beta > 0$ 调控率失真权衡： $$\mathcal{L} = \mathbb{E}_{z \sim q_\phi}[-\log p_\theta(\hat{x}|z)] + \beta D_{\mathrm{KL}}(q_\phi(z|x)\|p(z))$$

  • $\beta = 1$：标准 VAE
  • $\beta > 1$：更强信息瓶颈，潜变量更解耦，但重建质量下降
  • $\beta < 1$：重建质量优先

3. 课程总结与考核重点

3.1 必考内容 必考

期末闭卷考试（70%），提供公式：Jessen、log-sum、Fano、高斯分布统计度量。

试卷结构：选择+判断（40分） + 论述+证明+计算（60分）

  • 噪声信道编码定理证明：约 15/100 分，必须掌握完整证明
  • 编码计算：Kraft 不等式、Huffman 码、Hamming 码，10-15 分
  • 马尔可夫性：熵率、稳态分布、不可约性、非周期性，10-15 分
  • 计算论述：稳态分布、信道容量（KKT 条件）、高斯信道，10-15 分

不考内容：SMB 定理证明、Viterbi 算法、凹凸性证明过程

易错：率失真定理的几何解释（强典型集覆盖引理、失真球）需掌握；信源信道分离定理结论获取过程需掌握。Viterbi 不考。

3.2 核心公式汇总 重点

信息度量关系： $$H(p) = \log|\mathcal{X}| - D_{\mathrm{KL}}(p\|u), \quad I(X;Y) = D_{\mathrm{KL}}(p_{XY}\|p_X p_Y)$$

信道容量：$C^I = \max_{P_X} I(X;Y)$（对称信道用 KKT 条件求解）

率失真函数：$R(D) = R^I(D) = \min_{p(\hat{v}|v): \mathbb{E}[d] \leq D} I(V; \hat{V})$

信源信道分离：$N/n \leq C^I / R^I(D)$，等价于 $NR^I(D) \leq \log_2 M \leq nC^I$
