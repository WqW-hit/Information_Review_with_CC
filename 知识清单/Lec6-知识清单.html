本讲用平稳马尔可夫链作为平稳遍历信源的具体例子，给出 状态转移矩阵、稳态分布、熵率和遍历性的完整分析框架。

二、马尔可夫过程基础

2.1 马尔可夫性质 必考

马尔可夫过程：当前状态只与前一个状态有关，与更早历史无关：

$$p(X_n \mid X_{n-1}, X_{n-2}, \ldots, X_1) = p(X_n \mid X_{n-1})$$

等价地，联合概率可分解为：

$$p(X_n, X_{n-1}, \ldots, X_1) = p(X_1)\,P(X_2\mid X_1)\cdots P(X_n\mid X_{n-1})$$

易错：马尔可夫性是对条件分布的约束（条件只保留上一步），不等于独立。相邻两步之间仍有依赖关系。

2.2 条件独立性质 重点

马尔可夫链的核心性质：给定当前状态，过去与未来条件独立（Past and future are conditionally independent given the present）：

$$X_{-1} \perp X_1 \mid X_0 \quad\Longleftrightarrow\quad p(X_1, X_{-1} \mid X_0) = p(X_1\mid X_0)\,p(X_{-1}\mid X_0)$$

推导（利用马尔可夫性）：

$$p(X_1, X_{-1}\mid X_0) = \frac{p(X_1,X_0,X_{-1})}{p(X_0)} = \frac{p(X_{-1})\,p(X_0\mid X_{-1})\,p(X_1\mid X_0)}{p(X_0)} = p(X_{-1}\mid X_0)\,p(X_1\mid X_0)$$

2.3 平稳马尔可夫过程 必考

在马尔可夫性之上加平稳性：条件转移概率与时间步无关：

$$p(X_n\mid X_{n-1}) = p(X_0\mid X_{-1}) \quad \forall n$$

即转移概率 $P(X=s_j\mid X_{\text{prev}}=s_i)$ 对所有时刻相同——这是建立有限状态转移矩阵的前提。

平稳马尔可夫过程是最简单的非独立平稳信源模型，是 SMB 定理的典型具体例子。

三、状态转移矩阵

3.1 定义与构造 必考

设状态空间 $\mathcal{S}=\{s_1,s_2,\ldots,s_m\}$（即符号集 $\mathcal{X}$），定义 状态转移概率：

$$P_{ij} \;\triangleq\; p(X_0=s_j \mid X_{-1}=s_i)$$

将所有 $m^2$ 种转移概率排成 $m\times m$ 矩阵 $P$，称为状态转移矩阵（transition matrix）。

3.2 矩阵性质 重点

  • 非负性：$P_{ij}\geq 0$（概率）
  • 行归一：$\sum_j P_{ij}=1$（从状态 $i$ 出发，必然转移到某个状态）

注意：行表示"从哪来"，列表示"到哪去"——$P_{ij}$ 是从 $s_i$ 转移到 $s_j$ 的概率。

易错：列之和不一定为1，行之和才为1。P 是行随机矩阵（row-stochastic），不是列随机矩阵。

3.3 概率演化：Chapman-Kolmogorov 方程 重点

设 $p_{X_k}$ 为第 $k$ 步的概率行向量（$1\times m$），则：

$$p_{X_0} = p_{X_{-1}}\,P \qquad \text{（前进一步）}$$

$$p_{X_k} = p_{X_{-1}}\,P^{k+1} \qquad \text{（前进 $k+1$ 步）}$$

$P^k$ 称为 $k$ 步转移矩阵，其 $(i,j)$ 元 $[P^k]_{ij}=p(X_k=s_j\mid X_0=s_i)$。

四、稳态分布

4.1 定义 必考

稳态分布（stationary distribution）$\mu$ 是满足以下方程的概率行向量：

$$\mu = \mu\,P \quad\text{且}\quad \sum_i \mu_i = 1,\;\mu_i\geq 0$$

直觉：若初始分布为 $\mu$，则经过任意步转移后分布仍为 $\mu$——系统处于统计稳定状态。

若平稳马尔可夫过程的初始分布就是 $\mu$，则该过程是平稳过程（联合分布时移不变）。

4.2 细致平衡（Detailed Balance）重点

若分布 $\mu$ 满足细致平衡条件（对所有 $i,j$）：

$$\mu_i\,P_{ij} = \mu_j\,P_{ji}$$

则 $\mu$ 是稳态分布。证明：

$$\sum_i \mu_i P_{ij} = \sum_i \mu_j P_{ji} = \mu_j \sum_i P_{ji} = \mu_j\cdot 1 = \mu_j$$

即 $\mu P = \mu$。$\blacksquare$

易错：细致平衡是稳态分布的充分条件，不是必要条件。存在不满足细致平衡但仍有稳态分布的马尔可夫链（如有向环）。

4.3 例子：4 状态链 常考

状态空间 $\{s_1,s_2,s_3,s_4\}$，转移矩阵：

P = [0.8  0.2  0    0  ]
    [0    0    0.5  0.5]
    [0.5  0.5  0    0  ]
    [0    0    0.2  0.8]

解 $\mu P=\mu$（联立方程）：

$$\mu_1=\frac{5}{14},\quad\mu_2=\frac{1}{7},\quad\mu_3=\frac{1}{7},\quad\mu_4=\frac{5}{14}$$

验证：对称结构（$s_1\leftrightarrow s_4$，$s_2\leftrightarrow s_3$），$\mu_1=\mu_4$，$\mu_2=\mu_3$。

五、马尔可夫信源的熵率

5.1 熵率计算公式 必考

对平稳马尔可夫信源，利用马尔可夫性，$H(X_0\mid X_{-\infty}^{-1})=H(X_0\mid X_{-1})$，故：

$$H(\mathcal{X}) = H(X_0\mid X_{-1}) = \sum_i \mu_i \,H(X_0\mid X_{-1}=s_i) = -\sum_{i,j}\mu_i P_{ij}\log P_{ij}$$

推导步骤：

$$H(X_0\mid X_{-1}) = \sum_i p(X_{-1}=s_i)\,H(X_0\mid X_{-1}=s_i) = \sum_i\mu_i\sum_j P_{ij}\log\frac{1}{P_{ij}}$$

易错：这里用了 $p(X_{-1}=s_i)=\mu_i$（稳态分布），而不是任意初始分布；且利用马尔可夫性将无穷条件 $X_{-\infty}^{-1}$ 化简为仅条件于 $X_{-1}$。

5.2 例子计算 常考

对上述4状态链，$H(0.8,0.2) = -0.8\log 0.8 - 0.2\log 0.2 \approx 0.722$ bit，$H(0.5,0.5)=1$ bit：

$$H(\mathcal{X}) = \mu_1 H(0.8,0.2)+\mu_2 H(0.5,0.5)+\mu_3 H(0.5,0.5)+\mu_4 H(0.8,0.2)$$

$$= \frac{5}{14}\times 0.722+\frac{1}{7}\times 1+\frac{1}{7}\times 1+\frac{5}{14}\times 0.722 = \frac{5}{7}\times 0.722+\frac{2}{7}\times 1 \approx 0.80 \text{ bit/符号}$$

六、遍历性（Ergodicity）

6.1 遍历性的定义与意义 必考

平稳马尔可夫信源是遍历的（ergodic）当且仅当：

  1. 存在唯一稳态分布
  2. 从任意初始分布出发，均收敛到该稳态分布

遍历性保证了 SMB 定理中"时间均值→系综均值"的结论成立，是应用 Birkhoff 遍历定理的前提。

有限状态马尔可夫链的遍历性等价条件：不可约性 + 非周期性。

6.2 不可约性（Irreducibility）必考

不可约性：从任意状态出发，可以（经有限步）到达任意其他状态。形式化：

$$\forall i,j,\;\exists N>0 \;\text{ s.t. }\; [P^N]_{ij}>0$$

等价地：状态转移图中，对所有 $i,j$ 存在从 $s_i$ 到 $s_j$ 的有向路径。

Perron-Frobenius 定理的推论：若 $P$ 不可约，则稳态分布存在且唯一，且 $\mu_i>0$（所有状态概率为正）。

易错：不可约要求双向可达（从 $i$ 到 $j$ 且从 $j$ 到 $i$），有向环 $s_1\to s_2\to s_3\to s_1$ 是不可约的（各状态互通）。

6.3 非周期性（Aperiodicity）必考

状态 $s_i$ 的周期（period）定义为：

$$d(i) = \gcd\{k:\,[P^k]_{ii}>0\}$$

非周期性：所有状态的周期为 1（$d(i)=1$）。

关键结论（在不可约链中成立）：

  • 所有状态拥有相同的周期，故只需检查一个状态
  • 实用判定：若存在 $[P^k]_{ii}>0$（转移矩阵某对角元非零，即图中存在自环），则 $d(i)=1$，链非周期
易错：不可约不蕴含非周期——有向环 $s_1\to s_2\to s_3\to s_1$ 不可约但周期为3，不是遍历的。

6.4 综合：不可约+非周期 → 遍历 必考

条件保证反例（缺少时）
不可约（Irreducible）稳态分布唯一且各分量$>0$吸收态链：某状态出不去，无法从任意分布收敛
非周期（Aperiodic）任意初始分布收敛到稳态$s_1\to s_2\to s_1$ 振荡，不收敛到稳态
两者均满足遍历：时间均值 a.s. 等于系综均值——

对例子中的4状态链：转移图中 $s_1$ 有自环（$P_{11}=0.8>0$），故非周期；所有状态在图中互通，故不可约 → 遍历。
