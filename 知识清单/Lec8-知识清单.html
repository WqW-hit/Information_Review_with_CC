一、Huffman 算法构造

算法描述 必考

Huffman 码是求解整数约束下最优前缀码的贪婪算法（1952，David Huffman）。核心思想：

  1. 取概率最小的两个符号，赋予相同码长（最长），且码字仅在最后一位不同（一个末位为 0，另一个为 1）
  2. 将这两个符号合并为一个新符号，新概率 = 两者之和，重复上述步骤直到只剩一个符号

直观等价：将此过程画成二叉树，树叶即为码字；构造过程从叶到根（bottom-up）。

易错：每次选择概率「最小的两个」，相同概率时顺序可以任意选——Huffman 码可能不唯一，但平均编码长度相同。

二、Huffman 最优性证明

最优前缀码的三个性质 必考

设符号概率排序 $p_1\ge p_2\ge\cdots\ge p_r$，任意最优前缀码满足：

  1. 码长单调：$l_1\le l_2\le\cdots\le l_r$（概率大的符号码长不超过概率小的）
  2. 最小概率符号有最长、相等码长：$l_{r-1}=l_r=l_{\max}$
  3. 姊妹对（sister pair）：存在一个最优编码树，其中两个最小概率符号 $s_{r-1},s_r$ 是姊妹节点（父节点相同，码字仅末位不同）

性质3的证明思路：若 $c_{r-1}$ 和 $c_{r-3}$ 不是姊妹对，但码长相同，交换两者位置不改变总编码长度，但可使两个最小概率符号成为姊妹对。

融合-扩展论证（Reduction Argument） 必考

设有 $r$ 个符号的原问题 $U$，融合最小概率两符号 $s_{r-1},s_r$ 为 $s_{r-1}^*$（概率 $p_{r-1}+p_r$），得到 $r-1$ 个符号的子问题 $V$。

关键关系：$L_U = L_V + p_{r-1} + p_r$

因此：

  • 若 $L_U$ 是 $U$ 的最优编码长度，则 $L_V$ 是 $V$ 的最优编码长度（否则可改进 $V$ 进而改进 $U$）
  • 反之，若 $L_V$ 是 $V$ 的最优，则 $L_U=L_V+p_{r-1}+p_r$ 是 $U$ 的最优

原问题与子问题具有相同的最优性结构，可用递归证明 Huffman 最优。

递归完整证明 重点

  • 基础情形：$r=2$，编码 $\{0,1\}$ 是唯一前缀码，显然最优
  • 递归步骤：对 $r$ 个符号，Huffman 先融合最小两符号 --> 对 $r-1$ 个符号子问题归纳（子问题最优）--> 扩展回 $r$ 个符号，$L_U=L_V+p_{r-1}+p_r$ 仍最优

三、Huffman 码 vs Shannon 码

整数优化视角 必考

两者都在解：$\min L=\sum p(x)l(x)$，$\text{s.t.}\sum 2^{-l(x)}\le 1$

编码方案码长选取性质
连续松弛最优$l^*(x)=-\log p(x)$（可能为分数）$L^*=H(X)$，不可实现
Shannon 码$l(x)=\lceil-\log p(x)\rceil$（整数）满足 $H(X)\le L
Huffman 码贪婪算法给出整数码长整数约束下的精确最优解
易错：$L_\text{Huffman}\le L_\text{Shannon}$ 是平均意义上的，不代表每个符号的码长 $l_\text{Huffman}(x)\le l_\text{Shannon}(x)$。

Huffman 码的性能界 重点

平均编码长度满足：

$$H(X) \le L_\text{Huffman} < H(X)+1$$

Gallager 冗余界（1978）：

$$L_\text{Huffman} \le H(X) + p_{\max} + 0.086$$

其中 $p_{\max}$ 是最大符号概率，常数 $0.086 = 1-\log_2 e + \log_2(\log_2 e) \approx 0.08607$。

比基础界 $H+1$ 更紧，特别是当 $p_{\max}$ 小时效果显著。

四、块编码与 N 阶扩展信源

额外 1 bit 的问题与解法 重点

Huffman 码最坏每符号多 1 bit（上界 $H+1$），解法：块编码（N 阶扩展信源）。

将 $N$ 个符号看作一个「超级符号」，对长为 $N$ 的序列整体做 Huffman 编码：

$$H(X) \le \frac{L_N}{N} < H(X)+\frac{1}{N}$$

代价：

  • 牺牲即时可译性（存在延时 $N$）
  • 需计算联合概率 $p(X^N)$，编码树中大量字符串几乎不出现（根据大数定理）
  • 未考虑符号间的相关性（非独立同分布时性能变差）

五、Huffman 与 YES/NO 决策树

等价性 常考

Huffman 码的二叉树对应一种最优的 YES/NO 问题策略：

  • 每问一个集合类问题（$X\in S$？），YES/NO 对应二叉树的左右分支
  • 最少平均问题数 = Huffman 码平均编码长度 $L^*$
  • 理论下界 = $H(X)$（Shannon 熵）

例（5个事件，概率 0.25, 0.25, 0.20, 0.15, 0.15）：Huffman 编码 10, 01, 00, 111, 110，平均问题数 $= 2\times(0.25+0.25)+2\times 0.20+3\times(0.15+0.15) = 2.30$。

六、互信息的凹凸性（作业内容）

互信息关于 $p(X)$ 是凹函数 重点

固定 $p(Y|X)$，$I(X;Y)$ 是 $p(X)$ 的凹函数。

证明思路：

  • $I(X;Y)=H(Y)-H(Y|X)$
  • $H(Y|X)=\sum_x p(x)H(Y|X=x)$ 是 $p(X)$ 的线性函数
  • 固定 $p(Y|X)$，$p(y)=\sum_x p(x)p(y|x)$ 是 $p(X)$ 的线性函数
  • $H(Y)$ 是 $p(Y)$ 的凹函数；凹函数与线性函数的复合仍为凹函数
  • 故 $H(Y)$ 是 $p(X)$ 的凹函数，而 $H(Y|X)$ 是线性的，因此 $I=H(Y)-H(Y|X)$ 是凹函数

互信息关于 $p(Y|X)$ 是凸函数 重点

固定 $p(X)$，$I(X;Y)$ 是 $p(Y|X)$ 的凸函数。

证明思路（利用 KL 散度的凸性）：

  • $I(X;Y)=D_{KL}(p(x)p(y|x)\,\|\, p(x)\sum_x p(x)p(y|x))$
  • $p_X\cdot(\lambda p^{(1)}_{Y|X}+(1-\lambda)p^{(2)}_{Y|X})=\lambda p^{(1)}_{XY}+(1-\lambda)p^{(2)}_{XY}$
  • 利用 KL 散度对联合分布的凸性：$D_{KL}(\lambda P_1+(1-\lambda)P_2\|Q)\le\lambda D_{KL}(P_1\|Q_1)+(1-\lambda)D_{KL}(P_2\|Q_2)$

辅助引理：凸函数与线性函数的复合 了解

若 $f$ 为凸（凹）函数，$\ell(x)=Ax+b$ 为线性函数，则 $f\circ\ell$ 仍为凸（凹）函数。

证明：$f\circ\ell(\lambda x_1+(1-\lambda)x_2)=f(\lambda(Ax_1+b)+(1-\lambda)(Ax_2+b))\le\lambda f(Ax_1+b)+(1-\lambda)f(Ax_2+b)$。
