本章整体概览

本章是信息论的核心基础，建立了一套完整的信息度量体系。 从单个事件的自信息量出发，经由公理化推导确立对数形式， 再扩展到随机变量级别的信息熵，进而刻画两个随机变量之间的 条件熵、互信息与联合熵， 最后引入衡量分布差异的KL散度与交叉熵。 这些概念构成了信息论后续所有定理（信源编码、信道容量等）的数学语言。

学习主线： 为什么需要量化信息 --> 如何量化单个事件 --> 如何量化随机变量 --> 如何量化两变量关系 --> 如何量化分布差异

前置知识与记号系统

随机变量与概率分布的记号 重点

是什么：本课程使用如下统一记号，熟悉记号是读懂公式的前提。

  • X：离散随机变量，取值集合（字母表）记为 $\mathcal{X}$
  • p(x) 或 $p_X(x)$：X 取值 x 的概率，即 $\Pr(X=x)$
  • p(x,y)：联合概率 $\Pr(X=x, Y=y)$
  • p(x|y)：条件概率 $\Pr(X=x \mid Y=y)$
  • 期望算子 $\mathbb{E}[\cdot]$：$\mathbb{E}[f(X)] = \sum_{x \in \mathcal{X}} p(x) f(x)$

为什么：后续所有度量都是概率的函数，记号统一后可以简洁地用期望形式书写公式。

易错：p(x) 表示"概率值"（一个数），不是"概率分布"（一个函数）。写 $H(X)$ 时 X 是随机变量，写 $H(p)$ 时 p 是分布；两者等价但含义侧重不同。

逐模块深度讲解

模块一：为什么需要定量衡量信息 重点

承上启下关系：Shannon 1948 年提出信息论，目标是给"信息"一个与语义无关的数学定义，从而研究通信系统的极限。

是什么：我们希望一个函数 $I(x)$ 能衡量事件 $x$ 发生所携带的"信息量"（surprise/不确定性的消除量）。

为什么：直觉上：

  • 越不可能发生的事，发生后给我们的信息越多（稀奇 --> 信息大）
  • 概率为 1 的确定事件，发生后无任何信息（没有惊喜 --> 信息量为 0）
  • 两个独立事件同时发生，信息量应该等于各自之和（可加性）

怎么用：这三条直觉将被精确化为三条公理，从而唯一确定 $I(x)$ 的函数形式。

模块二：自信息量的公理化推导 必考

是什么：自信息量（self-information）$I(x)$ 是事件 $x$ 发生时所含信息量。

三条公理：

  1. 单调递减性：$I(x)$ 是 $p(x)$ 的单调递减函数——概率越大，信息量越小。
  2. 连续性：$I(x)$ 是 $p(x)$ 的连续函数——概率的微小变化不应引起信息量的跳变。
  3. 可加性（独立性公理）：若 $X$ 与 $Y$ 相互独立，则 $$I(x, y) = I(x) + I(y)$$ 即 $f(p_1 \cdot p_2) = f(p_1) + f(p_2)$，其中 $f = I \circ p^{-1}$。

推导：满足 $f(ab) = f(a) + f(b)$（Cauchy 函数方程）且连续的实函数唯一形式为 $f(t) = c \log t$（$c$ 为常数）。因此： $$I(x) = -c \log p(x)$$ 取 $c=1$，底数选 2 时单位为 比特（bit），底数选 $e$ 时单位为 奈特（nat），底数选 10 时单位为 哈特（hart）。

最终定义： $$\boxed{I(x) = -\log_2 p(x) \quad [\text{bit}]}$$

注意点：

  • $p(x) = 1 \Rightarrow I(x) = 0$（必然事件无信息）
  • $p(x) \to 0 \Rightarrow I(x) \to +\infty$（极稀有事件信息量无限大）
  • $0 \le p(x) \le 1 \Rightarrow I(x) \ge 0$（自信息量非负）
易错：可加性公理要求的是两个独立事件，不是任意两个事件。对相关事件，$I(x,y) \ne I(x)+I(y)$。

模块三：信息量的单位换算 高频

是什么：三种单位由对数底数决定，互换公式由换底公式给出。

单位底数名称1 bit 等于
bit（比特）2二进制单位1 bit
nat（奈特）$e$自然单位$\ln 2 \approx 0.693$ nat
hart（哈特）10十进制单位$\log_{10} 2 \approx 0.301$ hart

换算公式：$I_{\text{bit}} = I_{\text{nat}} \times \log_2 e \approx I_{\text{nat}} \times 1.4427$

1 nat = $\log_2 e \approx 1.443$ bits，因为 $-\ln p = -\log_2 p \cdot \ln 2$，所以 nat 值 = bit 值 × $\ln 2$，即 bit 值 = nat 值 × $\log_2 e$。

易错：考题常考"1 nat 等于多少 bit"，答案是 $\log_2 e \approx 1.443$，不是 $\ln 2 \approx 0.693$。记住方向：nat 比 bit 大（更粗粒度），所以 1 nat > 1 bit。

模块四："白马非马"困境——熵不等于信息量 重点

承上启下关系：自信息量 $I(x)$ 依赖于具体的 $x$（哪个事件发生了）。但在通信/压缩中，我们在发送前并不知道 $x$ 是什么，需要一个对整个随机变量的"平均"度量。

是什么：$I(x)$ 是对特定实现 $x$ 的度量；信息熵 $H(X)$ 是对整个随机变量 $X$（其概率分布）的度量。

为什么区分：若 $X \sim p$，则在观测 $X$ 之前，我们对 $X$ 的"平均不确定性"是： $$H(X) = \mathbb{E}_X[I(X)] = \mathbb{E}_X[-\log p(X)]$$ 这与任何具体的 $x$ 值无关，完全由分布 $p$ 决定。

易错：$I(x)$ 是一个具体的数（某次实验结果的信息量）；$H(X)$ 是一个期望（整个分布的平均不确定性）。$H(X)$ 不依赖于 $X$ 取什么具体值，只依赖于概率分布 $p$。

模块五：信息熵的定义与基本性质 必考

是什么：

$$\boxed{H(X) = -\sum_{x \in \mathcal{X}} p(x) \log_2 p(x) = \mathbb{E}\!\left[\log_2 \frac{1}{p(X)}\right] \quad [\text{bit}]}$$

约定 $0 \log 0 = 0$（因为 $\lim_{p \to 0^+} p \log p = 0$）。

基本性质：

  • 非负性：$H(X) \ge 0$，等号当且仅当 $X$ 是确定性变量（某个 $p(x)=1$）。
  • 最大熵：在 $|\mathcal{X}|=N$ 时，$H(X) \le \log_2 N$，等号当且仅当 $X$ 均匀分布。
  • 凹性：$H(p)$ 是概率向量 $p$ 的凹函数。
  • 对称性：$H$ 关于概率向量的分量置换不变（与随机变量取什么值无关，只依赖概率值集合）。

特殊情形：二元随机变量 $X \in \{0,1\}$，$p(X=1)=p$： $$H_b(p) = -p \log_2 p - (1-p)\log_2(1-p)$$ 在 $p=0.5$ 时取最大值 1 bit；$p=0$ 或 $p=1$ 时取最小值 0。

怎么用：计算熵时，将所有非零概率项代入公式求和；为零的项直接忽略。

易错：熵是"不确定性"的度量，不是"信息量"的度量——接收信息后不确定性降低，因此熵减少量才是所获信息量。

模块六：信息增益——接收信息等于熵减 必考

是什么：观测到事件 $Y=y$ 之后，对 $X$ 不确定性的减少量： $$\text{信息增益}(Y=y \to X) = H(X) - H(X \mid Y=y)$$

为什么：信息的本质是"不确定性的消除"。通信的过程正是通过接收信号 $Y$，减少对原始信息 $X$ 的不确定性。

怎么用：对 $y$ 取期望即得互信息 $I(X;Y) = H(X) - H(X|Y)$（见模块八）。

模块七：条件熵——观测Y后X的剩余不确定性 必考

是什么：条件熵是在已知 $Y$ 的条件下，$X$ 的平均不确定性： $$\boxed{H(X|Y) = \sum_{y} p(y) \, H(X|Y=y) = -\sum_{x,y} p(x,y) \log_2 p(x|y)}$$

推导： $$H(X|Y) = \mathbb{E}_Y[H(X|Y=y)] = \mathbb{E}_Y\!\left[-\sum_x p(x|y)\log p(x|y)\right] = -\mathbb{E}_{X,Y}[\log p(X|Y)]$$

重要性质：

  • 条件不增熵：$H(X|Y) \le H(X)$（观测 $Y$ 平均上不会增加对 $X$ 的不确定性）。
  • 等号成立当且仅当 $X$ 与 $Y$ 相互独立。
  • $H(X|X) = 0$（知道 $X$ 本身后没有不确定性）。

注意点：$H(X|Y) \ne H(X|Y=y)$——前者是期望，后者是对特定 $y$ 的熵。

易错：条件熵 $H(X|Y)$ 非负（因为是概率加权平均，每项 $H(X|Y=y) \ge 0$），但单次条件信息量 $\log p(x|y)/p(x)$ 可正可负。

模块八：互信息——量化两变量的关联程度 必考

是什么：互信息衡量 $X$ 和 $Y$ 之间共享的信息量： $$\boxed{I(X;Y) = H(X) - H(X|Y) = H(Y) - H(Y|X)}$$

三种等价表达：

  1. $I(X;Y) = H(X) - H(X|Y)$ Y 给 X 带来的信息量
  2. $I(X;Y) = H(X) + H(Y) - H(X,Y)$ 联合熵减去边缘熵之和的冗余量
  3. $I(X;Y) = D_{\mathrm{KL}}(p(x,y) \| p(x)p(y))$ 联合分布与独立分布的 KL 散度

重要性质：

  • 对称性：$I(X;Y) = I(Y;X)$（证明：由表达式 2 明显）
  • 非负性：$I(X;Y) \ge 0$（证明：由表达式 3 + KL 散度非负性）
  • $I(X;Y) = 0 \Leftrightarrow X \perp Y$（独立 --> 互信息为零）
  • $I(X;X) = H(X)$（X 关于自身的互信息 = 熵）
易错：互信息 $I(X;Y)$ 一定非负，但"单次互信息"（点互信息）$i(x;y) = \log \frac{p(x,y)}{p(x)p(y)}$ 可以为负（例如当 $p(x|y) < p(x)$ 时）。

模块九：联合熵与链式法则 必考

是什么：联合熵是两个随机变量联合分布的熵： $$H(X,Y) = -\sum_{x,y} p(x,y) \log_2 p(x,y)$$

链式法则（推导）：利用 $p(x,y) = p(x) p(y|x)$， $$H(X,Y) = -\mathbb{E}[\log p(X,Y)] = -\mathbb{E}[\log p(X)] - \mathbb{E}[\log p(Y|X)] = H(X) + H(Y|X)$$ $$\boxed{H(X,Y) = H(X) + H(Y|X) = H(Y) + H(X|Y)}$$

推广（多变量链式法则）： $$H(X_1, X_2, \ldots, X_n) = \sum_{i=1}^n H(X_i \mid X_1, \ldots, X_{i-1})$$

上下界： $$\max(H(X), H(Y)) \le H(X,Y) \le H(X) + H(Y)$$ 右侧等号当且仅当 $X \perp Y$；左侧等号当某变量是另一变量的确定性函数时取到。

易错：$H(X,Y) = H(X) + H(Y)$ 仅在独立时成立，一般情况下是小于等于号。

模块十：KL散度（相对熵）——分布之间的"距离" 重点

是什么：KL散度（Kullback-Leibler divergence）又称相对熵，衡量用分布 $q$ 来近似分布 $p$ 时的"损失"： $$\boxed{D_{\mathrm{KL}}(p \| q) = \sum_x p(x) \log_2 \frac{p(x)}{q(x)} = \mathbb{E}_p\!\left[\log \frac{p(X)}{q(X)}\right]}$$

约定：若 $q(x)=0$ 而 $p(x)>0$，则该项为 $+\infty$；$0 \log 0/0 = 0$。

性质：

  • 非负性：$D_{\mathrm{KL}}(p\|q) \ge 0$，等号当且仅当 $p = q$（由 Jensen 不等式证明，$\log$ 为凹函数）。
  • 不对称性：$D_{\mathrm{KL}}(p\|q) \ne D_{\mathrm{KL}}(q\|p)$（一般情况下）。
  • 无三角不等式：不满足 $D_{\mathrm{KL}}(p\|r) \le D_{\mathrm{KL}}(p\|q) + D_{\mathrm{KL}}(q\|r)$。

为什么不是"距离"：真正的距离度量需满足（1）非负性、（2）对称性、（3）三角不等式。KL 散度只满足（1），故称"伪距离"或"散度"而非"距离"。

与互信息的联系： $$I(X;Y) = D_{\mathrm{KL}}(p(x,y) \| p(x)p(y))$$ 互信息是联合分布与独立乘积分布之间的 KL 散度。

易错：KL 散度有方向性——$D_{\mathrm{KL}}(p\|q)$ 是"用 q 拟合 p 时"的代价，与"用 p 拟合 q 时"不同。在机器学习中，$p$ 通常是真实分布，$q$ 是模型分布。

模块十一：交叉熵及其在机器学习中的应用 重点

是什么：交叉熵是用分布 $q$ 编码分布 $p$ 的平均码长： $$\boxed{H(p,q) = CE(p\|q) = -\sum_x p(x) \log_2 q(x) = \mathbb{E}_p\!\left[-\log q(X)\right]}$$

与熵和 KL 散度的关系： $$CE(p\|q) = H(p) + D_{\mathrm{KL}}(p\|q)$$

推导：$CE(p\|q) = -\sum p \log q = -\sum p\log p + \sum p \log(p/q) = H(p) + D_{\mathrm{KL}}(p\|q)$。

为什么在机器学习中常用：训练分类模型时，$p$ 是标签的真实分布（固定的 one-hot），$q$ 是模型输出的预测分布（可学习）。由于 $H(p)$ 与模型参数无关，最小化 $CE(p\|q)$ 等价于最小化 $D_{\mathrm{KL}}(p\|q)$，即让模型分布尽量接近真实分布。

二分类交叉熵（常考）：$CE = -y \log \hat{y} - (1-y)\log(1-\hat{y})$

易错：$CE(p\|q) \ne CE(q\|p)$（交叉熵不对称）。$CE(p\|p) = H(p)$（用真实分布编码自身 = 熵）。

模块十二：Venn图——各度量的几何关系 必考

是什么：下表总结了本章六大度量的所有关系（Venn 图形式）：

关系式直觉解释
$H(X,Y) = H(X) + H(Y|X)$联合 = X 的不确定性 + 知 X 后 Y 的剩余不确定性
$H(X,Y) = H(Y) + H(X|Y)$对称地，联合 = Y 的不确定性 + 知 Y 后 X 的剩余
$I(X;Y) = H(X) - H(X|Y)$互信息 = 观测 Y 后 X 减少的不确定性
$I(X;Y) = H(X) + H(Y) - H(X,Y)$互信息 = 两个圆圈面积之和 - 联合面积（交集）
$H(X,Y) = H(X) + H(Y) - I(X;Y)$联合 = 两边缘熵之和 - 重叠部分（互信息）
$H(X|Y) = H(X,Y) - H(Y)$条件熵 = 联合熵 - 已知量的熵

补充解释（Venn 图描述）：将 $H(X)$ 画为左圆，$H(Y)$ 画为右圆：

  • 左圆独占区域 = $H(X|Y)$（X 独有的不确定性）
  • 右圆独占区域 = $H(Y|X)$（Y 独有的不确定性）
  • 两圆交集 = $I(X;Y)$（共享信息）
  • 两圆总面积 = $H(X,Y)$（联合不确定性）
易错：Venn 图中的"交集"是互信息 $I(X;Y)$，不是 $H(X) \cap H(Y)$（后者没有定义）。考试画 Venn 图时，务必标出每个区域对应的量。

常见误区与易混淆点

混淆点正确理解
$I(x)$ vs $H(X)$前者是单次事件的信息量（依赖具体 $x$），后者是整个分布的平均不确定性（只依赖 $p$）
点互信息 vs 互信息点互信息 $i(x;y) = \log\frac{p(x,y)}{p(x)p(y)}$ 可正可负；互信息 $I(X;Y) = \mathbb{E}[i(X;Y)] \ge 0$
$H(X|Y)$ vs $H(X|Y=y)$前者是对 $y$ 的期望（一个数），后者是给定特定 $y$ 后的熵（也是一个数，但不同）
KL散度 vs 距离KL散度不对称、无三角不等式，不是度量意义上的距离
交叉熵 vs KL散度$CE(p\|q) = H(p) + D_{\mathrm{KL}}(p\|q)$，$p$ 固定时两者等价最小化
$H(X,Y) \le H(X)+H(Y)$ vs 等号等号仅在 $X \perp Y$ 时成立，不独立时严格小于

本章总结

本章建立了信息论的度量语言。核心逻辑链为：

  1. 公理化推导 --> $I(x) = -\log p(x)$（自信息量）
  2. 取期望 --> $H(X) = \mathbb{E}[I(X)]$（信息熵 = 平均不确定性）
  3. 条件化 --> $H(X|Y)$（观测 Y 后的剩余不确定性）
  4. 差值 --> $I(X;Y) = H(X)-H(X|Y)$（共享信息）
  5. 乘法分解取期望 --> $H(X,Y) = H(X)+H(Y|X)$（链式法则）
  6. 分布比较 --> $D_{\mathrm{KL}}(p\|q)$（分布差异）
  7. 分解 --> $CE(p\|q) = H(p) + D_{\mathrm{KL}}(p\|q)$（交叉熵）

掌握所有关系式的推导（而不仅仅是背公式），是信息论考试的核心要求。 Venn 图是这些关系的可视化工具，每次推导前先画图有助于确认方向。
