一、微分熵

定义 必考

微分熵（differential entropy）是离散熵在连续分布上的类比，定义为：

$$h(X) = \mathbb{E}[-\log f_X(x)] = -\int f_X(x)\log f_X(x)\,dx$$

类比定义还有：

  • 条件微分熵：$h(X|Y) = -\iint f_{X,Y}(x,y)\log f_{X|Y}(x|y)\,dx\,dy$
  • 联合微分熵：$h(X,Y) = -\iint f_{X,Y}(x,y)\log f_{X,Y}(x,y)\,dx\,dy$
易错：微分熵可以为负数！均匀分布 $X\sim U(a,b)$：$h(X)=\log(b-a)$，当 $b-a<1$ 时为负。

与离散熵的关系（量化） 重点

对连续分布以精度 $\Delta$ 量化：

$$H(X^\Delta) \approx h(X) + \log\frac{1}{\Delta}$$

当 $\Delta\to 0$ 时 $H(X^\Delta)\to\infty$，连续/模拟信号熵无穷大，微分熵本身不代表"实际比特数"。

若 $\Delta=2^{-n}$，则 $H(X^\Delta)\approx h(X)+n$，解读：将 $X$ 量化到 $n$ bit精度所需的平均比特数约为 $h(X)+n$。

变量替换性质 重点

对可逆线性变换 $Y=AX+c$（$\det A\ne 0$）：

$$h(AX+c) = h(X) + \log|\det A|$$

特例：$h(X+c)=h(X)$（平移不变）；$h(aX)=h(X)+n\log|a|$（缩放增加熵）。

证明关键：利用变量替换公式 $f_Y(y)=\frac{1}{|\det A|}f_X(A^{-1}(y-c))$，$dy=|\det A|\,dx$。

二、KL 散度与互信息（连续版）

连续 KL 散度 必考

$$D_{KL}(f\|g) = \int f(x)\log\frac{f(x)}{g(x)}\,dx \ge 0$$

非负性用 Jensen 不等式证明：$-\log$ 是凸函数，$\mathbb{E}[-\log\frac{g(x)}{f(x)}]\ge -\log\mathbb{E}\frac{g(x)}{f(x)}=-\log 1=0$。

连续互信息 必考

$$I(X;Y) = D_{KL}(f_{XY}\|f_X f_Y) = \iint f_{X,Y}(x,y)\log\frac{f_{X,Y}(x,y)}{f_X(x)f_Y(y)}\,dx\,dy$$

与微分熵关系（与离散情况相同）：

  • $I(X;Y)=h(X)-h(X|Y)=h(Y)-h(Y|X)=h(X)+h(Y)-h(X,Y)$
  • $h(X|Y)\le h(X)$（条件不增加熵）
  • 互信息仍然非负（来自 KL 散度非负性）
易错：互信息非负，但微分熵可以为负；互信息的表达式中微分熵可相互抵消负号。

三、高斯分布的统计度量

高斯分布的微分熵 必考

$n$ 维高斯 $X^n\sim\mathcal{N}(\boldsymbol{\mu},\boldsymbol{\Sigma})$：

$$h(X^n) = \frac{n}{2}\log(2\pi e) + \frac{1}{2}\log|\boldsymbol{\Sigma}|$$

证明思路：令 $Y^n=\Sigma^{-1/2}(X^n-\boldsymbol{\mu})$，则 $Y^n\sim\mathcal{N}(\mathbf{0},I)$；利用变量替换性质 $h(X^n)=h(Y^n)+\log|\det\Sigma^{1/2}|$；再计算标准高斯的微分熵，用 $\operatorname{tr}\mathbb{E}[yy^T]=\operatorname{tr}I=n$。

特例（一维）：$X\sim\mathcal{N}(\mu,\sigma^2)$，$h(X)=\frac{1}{2}\log(2\pi e\sigma^2)$。

高斯分布是最大微分熵分布 必考

在固定均值 $\boldsymbol{\mu}$ 和协方差 $\boldsymbol{\Sigma}$ 的约束下，高斯分布具有最大微分熵：

$$h(X^n) \le \frac{n}{2}\log(2\pi e) + \frac{1}{2}\log|\boldsymbol{\Sigma}|$$

等号当且仅当 $X^n\sim\mathcal{N}(\boldsymbol{\mu},\boldsymbol{\Sigma})$。

证明：设真实分布为 $f$，高斯为 $g$；利用 $D_{KL}(f\|g)\ge 0$，即 $-h(X^n)+\mathbb{E}_f[-\log g(x)]\ge 0$；计算 $\mathbb{E}_f[-\log g(x)]$，用 $\operatorname{tr}(\mathbb{E}_f[(x-\mu)(x-\mu)^T]\Sigma^{-1})=\operatorname{tr}(\Sigma\Sigma^{-1})=n$。

易错：约束是固定协方差矩阵（非固定方差的情况更弱），此时高斯最优。功率约束信道容量即由此得出。

互信息与线性相关性 重点

对联合高斯 $(X_1,X_2)\sim\mathcal{N}$，相关系数为 $\rho$：

$$I(X_1;X_2) = -\frac{1}{2}\log(1-\rho^2)$$

$I(X_1;X_2)=0 \Leftrightarrow \rho=0$（线性不相关 $\Leftrightarrow$ 独立）。

注意：对一般（非高斯）分布，线性不相关不等于独立；只有高斯分布时两者等价。

两高斯之间的 KL 散度 重点

$p=\mathcal{N}(\boldsymbol{\mu}_1,\boldsymbol{\Sigma}_1)$，$q=\mathcal{N}(\boldsymbol{\mu}_2,\boldsymbol{\Sigma}_2)$：

$$D_{KL}(p\|q) = \frac{1}{2}\left(\ln\frac{|\boldsymbol{\Sigma}_2|}{|\boldsymbol{\Sigma}_1|} - n + \operatorname{tr}(\boldsymbol{\Sigma}_2^{-1}\boldsymbol{\Sigma}_1) + (\boldsymbol{\mu}_1-\boldsymbol{\mu}_2)^T\boldsymbol{\Sigma}_2^{-1}(\boldsymbol{\mu}_1-\boldsymbol{\mu}_2)\right)$$

记忆要点：包含四项——对数行列式比、维度 $n$、迹项、马氏距离项。

四、高斯信道

信道模型与功率约束 必考

高斯信道模型：$Y_i = X_i + W_i$，$W_i\sim\mathcal{N}(0,\sigma^2)$，i.i.d.，与 $X_i$ 独立。

功率约束：$\frac{1}{n}\sum_{i=1}^n X_i^2 \le P$。

高斯信道容量推导 必考

$$C_I = \max_{p(X):\,\mathbb{E}[X^2]\le P} I(X;Y) = \frac{1}{2}\log\!\left(1+\frac{P}{\sigma^2}\right)$$

推导步骤：

  1. $I(X;Y) = h(Y) - h(Y|X) = h(Y) - h(W)$（因为 $Y|X=x$ 时 $Y=x+W$，$h(Y|X)=h(W)$）
  2. $h(W) = \frac{1}{2}\log(2\pi e\sigma^2)$（已知，固定）
  3. 最大化 $h(Y)$：$\operatorname{Var}(Y)=\operatorname{Var}(X)+\sigma^2\le P+\sigma^2$，由高斯最大熵定理 $h(Y)\le\frac{1}{2}\log(2\pi e(P+\sigma^2))$，等号当 $X\sim\mathcal{N}(0,P)$ 时取到
  4. $C_I = \frac{1}{2}\log(2\pi e(P+\sigma^2)) - \frac{1}{2}\log(2\pi e\sigma^2) = \frac{1}{2}\log\!\left(1+\frac{P}{\sigma^2}\right)$

最优输入分布：$X\sim\mathcal{N}(0,P)$，即满功率的高斯分布。

高斯信道的几何解释 常考

类比离散情况的噪声球：

  • 输出空间半径（$n$ 维球）：$\sqrt{n(P+\sigma^2)}$
  • 噪声球半径（每个码字对应的不确定范围）：$\sqrt{n\sigma^2}=\sqrt{n}\,\sigma$
  • 输入空间半径：$\sqrt{nP}$
  • 可区分球数：$\left(\frac{\sqrt{n(P+\sigma^2)}}{\sqrt{n}\,\sigma}\right)^n = \left(1+\frac{P}{\sigma^2}\right)^{n/2}$
  • $R = \frac{1}{n}\log\#\text{spheres} = \frac{1}{2}\log\!\left(1+\frac{P}{\sigma^2}\right)$

五、并行高斯信道与水填充

问题设置 必考

$n$ 个并行高斯信道，第 $i$ 个：$Y_i=X_i+W_i$，$W_i\sim\mathcal{N}(0,N_i)$，各信道噪声不同。

总功率约束 $\sum P_i=P$，最大化总信道容量：

$$\max_{\{P_i\}} \sum_{i=1}^n \frac{1}{2}\log\!\left(1+\frac{P_i}{N_i}\right), \quad \text{s.t.} \sum_{i=1}^n P_i = P,\; P_i\ge 0$$

水填充最优解（KKT 条件） 必考

写拉格朗日函数 $L$，对 $P_i$ 求导，加入 KKT 条件 $P_i\ge 0,\nu_i\ge 0,\nu_i P_i=0$：

$$P_i^* = (\mu - N_i)^+ = \max(\mu - N_i,\; 0)$$

其中水位线 $\mu = \frac{1}{2\lambda}$ 由约束 $\sum_{i=1}^n P_i^* = P$ 确定。

水填充直觉：把噪声 $N_i$ 看作第 $i$ 个池子的底部高度，将总功率 $P$ 像水一样从低到高注入，公共水位为 $\mu$；噪声太大（底部超过水位）的信道分配功率为 0。

易错：不是把功率平均分，也不是全给最好的信道。噪声小的信道分到更多功率，但有下限 0。

最优总容量：

$$C = \sum_{i:P_i^*>0} \frac{1}{2}\log\!\left(\frac{\mu}{N_i}\right)$$
