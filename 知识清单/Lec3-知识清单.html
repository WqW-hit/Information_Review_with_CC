模块一：凸函数与凹函数的定义

凸函数（Convex Function）的定义 必考

是什么：函数 $f$ 是凸函数，当且仅当对任意 $x_1, x_2$ 及 $0 \le \lambda \le 1$： $$\lambda f(x_1) + (1-\lambda) f(x_2) \ge f(\lambda x_1 + (1-\lambda) x_2)$$ 即：弦在函数图像之上（函数图像"开口向上"）。

等价条件（可微情形）：$f$ 是凸函数当且仅当 $f$ 始终在其任意切线之上： $$f(x) \ge f(x_0) + f'(x_0)(x - x_0), \quad \forall x, x_0$$

凹函数：$f$ 是凹函数 $\Leftrightarrow$ $-f$ 是凸函数，即： $$\lambda f(x_1) + (1-\lambda)f(x_2) \le f(\lambda x_1 + (1-\lambda)x_2)$$ 弦在函数图像之下（函数图像"开口向下"）。

常见函数的凹凸性（信息论高频使用）：

函数凹/凸记忆方法
$\log x$（$x>0$）凹函数图像开口向下；$(\log x)'' = -1/x^2 < 0$
$-\log x$（$x>0$）凸函数$(-\log x)'' = 1/x^2 > 0$
$x \log x$（$x>0$）凸函数$(x\log x)'' = 1/(x\ln 2) > 0$
$x^2$凸函数$(x^2)'' = 2 > 0$
$e^x$凸函数$(e^x)'' = e^x > 0$
$\sqrt{x}$（$x>0$）凹函数$(\sqrt{x})'' = -\frac{1}{4}x^{-3/2} < 0$
易错：$\log x$ 是凹函数（$\mathbb{E}[\log X] \le \log \mathbb{E}[X]$），$-\log x$ 是凸函数（$\mathbb{E}[-\log X] \ge -\log \mathbb{E}[X]$）。两者符号相反，Jensen 不等式方向也相反，务必先确认凹凸性再写不等号方向。

模块二：Jensen 不等式

Jensen 不等式的表述与推导 必考

是什么：设 $f$ 是凸函数，$X$ 是随机变量，则： $$\boxed{\mathbb{E}[f(X)] \ge f(\mathbb{E}[X])}$$ 等号成立当且仅当 $f$ 严格凸时 $X$ 是常数（即 $X$ 以概率 1 等于某常数）。

对凹函数方向反向：$\mathbb{E}[f(X)] \le f(\mathbb{E}[X])$。

推导思路：令 $x_0 = \mathbb{E}[X]$，由支撑超平面性质，存在 $n_0$ 使得： $$f(x) \ge f(x_0) + n_0^T(x - x_0), \quad \forall x$$ 两边对 $X$ 取期望： $$\mathbb{E}[f(X)] \ge f(x_0) + n_0^T(\mathbb{E}[X] - x_0) = f(\mathbb{E}[X]) + 0 = f(\mathbb{E}[X])$$

使用 Jensen 不等式的四步诀：

  1. 确认 $f$ 的凹凸性（凸 --> $\mathbb{E}[f] \ge f(\mathbb{E})$，凹 --> 反向）
  2. 明确"概率 $p$ 是什么"（谁在加权平均）
  3. 明确"$X$ 是什么"（期望作用在哪个变量上）
  4. 写出 $\mathbb{E}[f(X)]$ 和 $f(\mathbb{E}[X])$ 的展开式
易错：Jensen 不等式的左边是 $\mathbb{E}[f(X)]$（先算 $f$ 再取期望），右边是 $f(\mathbb{E}[X])$（先取期望再算 $f$）。对凸函数，"先算 $f$"更大。记住：凸函数"内大于外"（$f(\mathbb{E}) \le \mathbb{E}[f]$）。

Jensen 不等式的常见推论 重点

以下均为直接套用 Jensen 不等式的结果：

结论使用的 $f$凹/凸
$\mathbb{E}[X^2] \ge (\mathbb{E}[X])^2$（方差非负）$f(x)=x^2$凸
$\mathbb{E}[e^X] \ge e^{\mathbb{E}[X]}$（矩母函数下界）$f(x)=e^x$凸
$\mathbb{E}[\log X] \le \log \mathbb{E}[X]$（对数凹）$f(x)=\log x$凹
$\mathbb{E}[X\log X] \ge \mathbb{E}[X]\log\mathbb{E}[X]$$f(x)=x\log x$凸
$\mathbb{E}[-\log X] \ge -\log \mathbb{E}[X]$（KL非负性的基础）$f(x)=-\log x$凸

算术平均 ≥ 几何平均 ≥ 调和平均（AM-GM-HM 不等式）：令 $X$ 均匀取 $\{x_1,\ldots,x_n\}$，对严格凸函数 $f(x) = -\log x$ 用 Jensen： $$\mathbb{E}[-\log X] \ge -\log \mathbb{E}[X]$$ $$\Rightarrow \frac{1}{n}\sum_{i=1}^n (-\log x_i) \ge -\log\!\left(\frac{1}{n}\sum x_i\right)$$ $$\Rightarrow \frac{x_1+\cdots+x_n}{n} \ge \sqrt[n]{x_1 x_2 \cdots x_n}$$ 调和平均类似可得。

易错：AM≥GM 成立需要所有 $x_i > 0$，等号当且仅当所有 $x_i$ 相等（严格凸函数等号条件：$X$ 为常数）。

模块三：Log-sum 不等式

Log-sum 不等式的表述 必考

是什么：对任意 $a_i \ge 0$，$b_i \ge 0$（$i=1,\ldots,n$），有： $$\boxed{\sum_{i=1}^n a_i \log \frac{a_i}{b_i} \ge \left(\sum_{i=1}^n a_i\right) \log \frac{\sum_{i=1}^n a_i}{\sum_{i=1}^n b_i}}$$ 等号当且仅当 $a_i/b_i = \text{const}$（各比值相等）。

为什么重要：信息论中所有含 $\sum p \log(p/q)$ 结构的不等式（KL散度、熵的凹凸性、互信息凹凸性）都可以直接套用 log-sum 不等式，比 Jensen 路线更简洁。

Log-sum 不等式的两种证明路线 重点

路线一（$f(x) = \log x$ 凹函数 + Jensen）：

令 $p_i = b_i / \sum b_j$，$X_i = a_i/b_i$，则 log-sum 不等式等价于： $$\mathbb{E}_p[\log X] \le \log \mathbb{E}_p[X]$$ 即对凹函数 $\log x$ 的 Jensen 不等式。

路线二（$f(x) = x\log x$ 凸函数 + Jensen）：

令 $p_i = b_i/\sum b_j$，$X_i = a_i/b_i$，则 log-sum 不等式等价于： $$\mathbb{E}[X\log X] \ge \mathbb{E}[X] \log \mathbb{E}[X]$$ 即对凸函数 $x \log x$ 的 Jensen 不等式。

易错：两种路线本质相同，区别在于如何设置"概率 $p$"和"随机变量 $X$"。考试时选其中一种路线写清楚即可，不要混用。

Log-sum 不等式的核心应用：KL散度非负性 必考

令 $a_i = p(x_i)$，$b_i = q(x_i)$，则： $$D_{\mathrm{KL}}(p\|q) = \sum_x p(x)\log\frac{p(x)}{q(x)} \ge \left(\sum_x p(x)\right)\log\frac{\sum_x p(x)}{\sum_x q(x)} = 1 \cdot \log\frac{1}{1} = 0$$ 等号当且仅当 $p(x)/q(x) = \text{const}$，即 $p(x) = q(x)$ 对所有 $x$。

这是用 log-sum 不等式最简洁地证明 Gibbs 不等式（KL散度非负性）的方法。

模块四：KL散度非负性及其推论

Gibbs 不等式（KL散度非负性） 必考

$$\boxed{D_{\mathrm{KL}}(p\|q) \ge 0, \quad \text{等号} \Leftrightarrow p = q}$$

三条重要推论（均由 $D_{\mathrm{KL}} \ge 0$ 直接推出）：

  1. 互信息非负性：$I(X;Y) = D_{\mathrm{KL}}(p(x,y)\|p(x)p(y)) \ge 0$，等号 $\Leftrightarrow X \perp Y$
  2. 条件熵不大于原熵：$H(X|Y) \le H(X)$，等号 $\Leftrightarrow X \perp Y$
  3. 次可加性：$H(X) + H(Y) \ge H(X,Y)$，等号 $\Leftrightarrow X \perp Y$

推论2的证明：$H(X|Y) \le H(X) \Leftrightarrow I(X;Y) = H(X) - H(X|Y) \ge 0$。

推论3的证明：$H(X,Y) = H(X) + H(Y|X) \le H(X) + H(Y)$（因为 $H(Y|X) \le H(Y)$）。

易错：推论2（$H(X|Y) \le H(X)$）是期望层面的结论。对于特定观测值 $y$，$H(X|Y=y)$ 可以大于 $H(X)$；但对所有 $y$ 取期望后一定不超过 $H(X)$。

均匀分布熵最大 重点

结论：设 $|\mathcal{X}| = N$，则 $H(p) \le \log_2 N$，等号当且仅当 $p$ 是均匀分布 $u(x) = 1/N$。

关键恒等式： $$H(p) = \log N - D_{\mathrm{KL}}(p\|u)$$ 推导：$H(p) = -\sum p\log p = -\sum p\log p + \sum p\log(1/N) + \sum p\log N = \log N - D_{\mathrm{KL}}(p\|u)$

由 $D_{\mathrm{KL}}(p\|u) \ge 0$ 立得 $H(p) \le \log N$，等号当 $D_{\mathrm{KL}}=0$，即 $p=u$（均匀分布）。

直觉：越接近均匀分布（越"没有规律"），熵越大；信源越随机，所需编码越长。

模块五：KL散度的凹凸性

KL散度是 $(p,q)$ 的联合凸函数 重点

结论：$D_{\mathrm{KL}}(p\|q)$ 是关于 $(p, q)$ 的凸函数，即对 $0 \le \lambda \le 1$： $$\lambda D_{\mathrm{KL}}(p_1\|q_1) + (1-\lambda)D_{\mathrm{KL}}(p_2\|q_2) \ge D_{\mathrm{KL}}(\lambda p_1 + (1-\lambda)p_2 \| \lambda q_1 + (1-\lambda)q_2)$$

证明思路（用 log-sum 不等式）：令 $a_1 = \lambda p_1(x)$，$b_1 = \lambda q_1(x)$，$a_2 = (1-\lambda)p_2(x)$，$b_2 = (1-\lambda)q_2(x)$，直接套用 log-sum 不等式即得。

特殊情形（固定其中一个分布）：

  • 固定 $q$：$D_{\mathrm{KL}}(p\|q)$ 是 $p$ 的凸函数
  • 固定 $p$：$D_{\mathrm{KL}}(p\|q)$ 是 $q$ 的凸函数

模块六：熵的凹凸性

熵 $H(p)$ 是概率向量 $p$ 的凹函数 必考

结论：对 $0 \le \lambda \le 1$： $$\boxed{\lambda H(p_1) + (1-\lambda)H(p_2) \le H(\lambda p_1 + (1-\lambda)p_2)}$$ 即混合分布的熵 $\ge$ 各分布熵的混合（"混合增加不确定性"）。

三种证明路线（考试任选其一）：

  1. Log-sum 不等式路线：令 $a_1=\lambda p_1(x)$，$b_1=\lambda$，$a_2=(1-\lambda)p_2(x)$，$b_2=1-\lambda$，套用 log-sum，左边化简为 $-[\lambda H(p_1)+(1-\lambda)H(p_2)]$，右边化简为 $-H(\lambda p_1+(1-\lambda)p_2)$。
  2. Jensen 不等式路线（$f(x) = x\log x$ 凸函数）：同样设置 $p_i$ 和 $x_i$，套用 $\mathbb{E}[X\log X] \ge \mathbb{E}[X]\log\mathbb{E}[X]$。
  3. KL散度路线：利用 $H(p) = \log N - D_{\mathrm{KL}}(p\|u)$，由 KL 散度对 $p$ 的凸性，得 $-D_{\mathrm{KL}}(\cdot\|u)$ 是凹函数，即 $H(\cdot)$ 是凹函数。
  4. 条件熵路线（最优雅）：构造混合随机变量 $Y \in \{1,2\}$，$p(Y=1)=\lambda$，$X|Y=1 \sim p_1$，$X|Y=2 \sim p_2$，则 $X \sim \lambda p_1+(1-\lambda)p_2$，由 $H(X|Y) \le H(X)$，得 $\lambda H(p_1)+(1-\lambda)H(p_2) \le H(\lambda p_1+(1-\lambda)p_2)$。
易错：此处"凹"是相对于概率向量 $p$ 而言的（$p$ 生活在概率单纯形上），不是相对于 $x$ 值。两个概率分布 $p_1, p_2$ 的凸组合 $\lambda p_1 + (1-\lambda)p_2$ 仍是合法的概率分布（各分量求和为1）。

熵凹性的直觉 高频

"混合分布增加熵"（Mixing probability distributions increases uncertainty）：

将两个分布混合后，不确定性不减（至少不小于各分布不确定性的加权平均）。例如：将一枚均匀硬币（$H(p_1)=1$ bit）与一枚全正面的硬币（$H(p_2)=0$ bit）以 $\lambda=0.5$ 混合，得到 $p(正面)=0.75$ 的硬币，$H \approx 0.81$ bit $\ge 0.5 \times 1 + 0.5 \times 0 = 0.5$ bit。

模块七：互信息的凹凸性

互信息的凹凸性 重点

结论：互信息 $I(X;Y)$ 可以看作 $p(X)$ 和 $p(Y|X)$ 的函数：

  • 固定 $p(Y|X)$：$I(X;Y)$ 是关于 $p(X)$ 的凹函数
  • 固定 $p(X)$：$I(X;Y)$ 是关于 $p(Y|X)$ 的凸函数

凹性（固定信道 $p(Y|X)$）的意义：信道容量优化（$\max_{p(X)} I(X;Y)$）的目标函数是凸优化，存在全局最优解，且最优输入分布可以通过 KKT 条件求解（Blahut-Arimoto 算法的基础）。

展开式： $$I(X;Y) = \sum_{x,y} p(x)p(y|x)\log p(y|x) - \sum_{x,y} p(x)p(y|x)\log\sum_x p(x)p(y|x)$$ 第一项对 $p(X)$ 是线性的（凹），第二项通过 log-sum 不等式可验证对 $p(X)$ 是凸的，总体为凹。

易错：凹在 $p(X)$ 上（固定信道时），凸在 $p(Y|X)$ 上（固定输入时）。方向非对称，与熵的纯凹性不同。

模块八：本章不等式体系总览

Venn 图式不等式关系 必考

结论证明工具等号条件
$D_{\mathrm{KL}}(p\|q) \ge 0$Log-sum 或 Jensen（$-\log x$ 凸）$p = q$
$I(X;Y) \ge 0$$I = D_{\mathrm{KL}} \ge 0$$X \perp Y$
$H(X|Y) \le H(X)$$I(X;Y) \ge 0$$X \perp Y$
$H(X,Y) \le H(X)+H(Y)$$H(Y|X) \le H(Y)$$X \perp Y$
$H(p) \le \log|\mathcal{X}|$$H(p) = \log N - D_{\mathrm{KL}}(p\|u)$$p$ 为均匀分布
$H(p)$ 是 $p$ 的凹函数Log-sum / Jensen / KL凸性 / 条件熵路线（凸组合时等号：$p_1=p_2$）
$D_{\mathrm{KL}}(p\|q)$ 是 $(p,q)$ 的凸函数Log-sum 不等式$p_1/q_1 = p_2/q_2$ 各处
$I(X;Y)$ 关于 $p(X)$ 为凹熵凹性 + 条件熵凸性（见信道容量）
$I(X;Y)$ 关于 $p(Y|X)$ 为凸KL散度凸性（见信道编码）

常见证明思路总结

遇到信息论不等式的通用策略：

  1. 先尝试直接套 Log-sum 不等式（最省力，适合含 $\sum a_i \log(a_i/b_i)$ 的结构）
  2. 若不符合 log-sum 形式，尝试Jensen 不等式（先判断 $f$ 的凹凸性，再设 $p$ 和 $X$）
  3. 若结论是关于信息量关系（$H$, $I$, $D_{\mathrm{KL}}$），尝试转化为 KL 散度非负性
  4. 对凹凸性命题，考虑条件熵路线（构造辅助随机变量 $Y$）
易错：套 Jensen 时最常见的错误是写错不等号方向。口诀："凸函数，$\mathbb{E}[f] \ge f(\mathbb{E})$；凹函数，$\mathbb{E}[f] \le f(\mathbb{E})$"。
