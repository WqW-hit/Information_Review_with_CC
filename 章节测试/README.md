## 使用方式

1. 下载 `.html` 文件到本地
2. 用浏览器（推荐 Chrome / Edge）直接打开
3. 作答完毕后点击**提交答案**，即可查看得分、解析及错因提示

**功能说明：**

- 选择题和判断题提交后自动评分
- 简答/论述/综合题提供参考答案与推理过程，供自我对照
- 每题均附有**易错点（pitfall）**说明，帮助识别常见误区
